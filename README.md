# Another Alt Tab

[English README](./README-en.md)

`alt+tab`是Windows当中切换窗口的快捷键，然后这些窗口我只想放2-3个现在频繁使用的窗口，所以我可以轻易地使用`alt+tab`进行切换。但是我又有一个窗口专门管理笔记，一个窗口专门进行项目构造的绘图，是不是也会需要它们，如果全部放在`alt+tab`这里面的话，会扰乱正常的工作窗口的顺序，所以我使用ImGui和SDL3开发了这个简单的工具，相当于开辟了一个新的空间，两个窗口管理空间刚刚好，再细分也没必要了，反倒效率会变低。

## 如何编译

环境是MSVC的开发环境控制台，`python build.py all`即可编译，`python build.py main`则只编译`main.cpp`，而不编译ImGui的源文件，它们只需要编译一次即可

## 如何使用

运行exe后，托盘会多出来一个应用，右键可以选择退出，左键双击则会打开窗口管理器：

![](image/README0.png)

其中点击第一个按钮后鼠标会变为十字（离开工具窗口后就不是了，这个懒得改了，能看出现在是选择状态就行），这时候进入窗口选择状态，这时候点击一个窗口它就会把点击的窗口变为浮动工具栏，并进入到我们自己的窗口管理空间，点击了一次左键或者点击`esc`退出。

点击第二个按钮会将我们自己的窗口管理空间的最后一个窗口放回alt+tab列表

这个软件默认使用`alt+]`作为快捷键，然后使用`alt+]`就可以像`alt+tab`一样选择我们自己窗口管理空间当中的窗口打开。

以及程序在托盘右键退出的时候会把所有窗口放回`alt+tab`列表
